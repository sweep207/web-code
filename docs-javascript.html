⚡ JavaScript - Ngôn ngữ Lập trình Web

JavaScript là gì?

JavaScript là ngôn ngữ lập trình để tạo tính tương tác cho website. Mọi website hiện đại đều sử dụng JavaScript.

JavaScript có thể:

  • Thay đổi nội dung HTML
  • Thay đổi styles CSS
  • Xử lý sự kiện (click, hover, input...)
  • Gửi/nhận dữ liệu từ server
  • Tạo animations và effects

1. Biến (Variables)

// Khai báo biến
let name = "Alice";      // Có thể thay đổi
const age = 25;          // Không thể thay đổi (constant)
var city = "Hanoi";      // Cách cũ (không nên dùng)

// Kiểu dữ liệu
let number = 42;         // Number (số)
let text = "Hello";      // String (chuỗi)
let isTrue = true;       // Boolean (true/false)
let nothing = null;      // Null (rỗng)
let notDefined;          // Undefined (chưa gán giá trị)

2. Hàm (Functions)

// Function declaration
function greet(name) {
    return "Hello, " + name;
}

// Arrow function (ES6)
const greet = (name) => {
    return `Hello, ${name}`;  // Template literal
};

// Arrow function ngắn gọn
const add = (a, b) => a + b;

// Gọi function
let message = greet("Alice");
console.log(message);  // "Hello, Alice"
console.log(add(5, 3));  // 8

3. Điều kiện (If/Else)

let age = 18;

if (age >= 18) {
    console.log("Đủ tuổi");
} else if (age >= 13) {
    console.log("Thiếu niên");
} else {
    console.log("Trẻ em");
}

// Ternary operator (ngắn gọn)
let status = age >= 18 ? "Adult" : "Minor";

4. Vòng lặp (Loops)

// For loop
for (let i = 0; i < 5; i++) {
    console.log(i);  // 0, 1, 2, 3, 4
}

// While loop
let count = 0;
while (count < 3) {
    console.log(count);
    count++;
}

// Array forEach
const fruits = ["apple", "banana", "orange"];
fruits.forEach(fruit => {
    console.log(fruit);
});

// Array map (tạo array mới)
const numbers = [1, 2, 3, 4, 5];
const doubled = numbers.map(num => num * 2);
console.log(doubled);  // [2, 4, 6, 8, 10]

5. DOM Manipulation

// Lấy element
const element = document.getElementById('myId');
const elements = document.querySelectorAll('.myClass');

// Thay đổi nội dung
element.textContent = "New text";
element.innerHTML = "<strong>Bold text</strong>";

// Thay đổi style
element.style.color = "red";
element.style.fontSize = "20px";

// Thêm/xóa class
element.classList.add('active');
element.classList.remove('hidden');
element.classList.toggle('open');

// Event listener
element.addEventListener('click', () => {
    console.log('Clicked!');
});

// Tạo element mới
const newDiv = document.createElement('div');
newDiv.textContent = 'New element';
document.body.appendChild(newDiv);

6. Objects & Arrays

// Object
const person = {
    name: "Alice",
    age: 25,
    greet: function() {
        return `Hello, I'm ${this.name}`;
    }
};

console.log(person.name);     // "Alice"
console.log(person.greet());  // "Hello, I'm Alice"

// Array
const colors = ["red", "green", "blue"];
console.log(colors[0]);       // "red"
console.log(colors.length);   // 3

// Array methods
colors.push("yellow");        // Thêm vào cuối
colors.pop();                 // Xóa cuối
colors.shift();               // Xóa đầu
colors.unshift("purple");     // Thêm vào đầu
